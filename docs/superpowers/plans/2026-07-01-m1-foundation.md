# Milestone 1 — Foundation & Design System Implementation Plan

Goal: Stand up a tested React + TypeScript + Vite + Tailwind app for Title Run with the "Octagon Elite" theme and a navigable four-screen shell (walking skeleton).

Architecture: Client-only SPA. A pure design-tokens module is the single source for Tailwind theme values. A pure screen registry drives a lightweight useState-based navigation shell (no router). Placeholder screens render for each of the four surfaces. Vitest + React Testing Library provide the test harness; GitHub Actions runs typecheck + tests + build on every PR.

Tech Stack: React 18, TypeScript (strict), Vite 5, Tailwind CSS 3.4.x (TS config importing tokens), Vitest, @testing-library/react, jsdom.

Global Constraints:
- TypeScript strict mode on; Tailwind v3.4.x with theme.extend; no CSS-in-JS; no backend/network/router in M1.
- Fonts via Google Fonts <link> in index.html: Anton (display), Archivo Narrow (body), Space Mono (mono/stats), Material Symbols (icons).
- Design tokens come verbatim from docs/design/design-system.md; src/theme/tokens.ts is their single source and Tailwind imports from it.
- Screen ids exactly: championship-hub, draft, fight, reward. Default: championship-hub.
- Every task ends green (vitest run + vite build) then a commit.
